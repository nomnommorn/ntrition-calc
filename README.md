# 実量換算 - 栄養成分カリキュレーター

栄養成分表示は「100gあたり」など決まった基準量で書かれていることが多く、実際のパッケージの内容量（例：85g）とは異なる場合があります。
このアプリは、表示の基準量と実際の内容量を入力するだけで、エネルギー・たんぱく質・脂質・炭水化物などの栄養成分を実量換算してくれるシンプルなツールです。

## 使い方

1. 「表示の基準量」に栄養成分表示の基準（例：100）を入力
2. 「実際の内容量」に実際の重さ（例：85）を入力
3. 各項目の「基準値」欄にパッケージに書かれた数値を入力
4. 自動的に実量換算後の値が右側に表示されます

計算式はシンプルです。

```
実量換算値 = 基準値 ÷ 基準量 × 実際の内容量
```

項目は自由に追加・削除・名称変更でき、カロリー以外の任意の栄養素（食物繊維、糖質、ナトリウムなど）にも対応しています。

## セットアップ

Node.js（推奨: v18以上）と npm が必要です。

```bash
npm install
npm run dev
```

ブラウザで `http://localhost:5173` を開いてください。

## ビルド

```bash
npm run build
```

`dist/` フォルダに静的ファイルが出力されます。GitHub Pages や Netlify、Vercel などにそのままデプロイできます。

## GitHub Pages へのデプロイ例

```bash
npm run build
# dist フォルダの中身を gh-pages ブランチにプッシュ、
# または GitHub Actions / Pages の設定で dist を公開ディレクトリに指定
```
